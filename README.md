# Сервис для отправки уведомлений

Работат по RestAPI(решение тех-лида, я настаивал на event-driven architecture с использованием брокера сообщений). Кнтроллер для отправки писем можно заменить на услоный kafka-receiver.

Помимо реализации с 0 до прода занимался поиском оптимального облачного провайдера для корпоративной почты(в нашем случае TimewebCloud) и его настройкой
